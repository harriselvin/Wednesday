There are different programming languages that involve web development

HTML

HTML is the standard markup language for Web pages. You can created the structure of your Web page using HTML.

CSS

CSS is the language we use to style an HTML document. It describes how HTML elements should be displayed.

JavaScript

JavaScript is the programming language of the Web and also is the world's most popular programming language. It makes your Web page interactive.
